## ✦ Présentation

Ce repo contient ma **carte de contact numérique** — une page web légère, rapide et soignée, conçue pour être partagée via **NFC**, **QR Code**, ou simple lien URL.

Elle remplace la carte de visite papier avec un rendu moderne, interactif et toujours à jour.

---

## ✦ Fonctionnalités

| Élément | Description |
|---|---|
| 🎨 **Design glassmorphism** | Palette beige chaud · accents forêt & terra · grain texture |
| ✨ **Splash screen animé** | Carte LB avec entrée 3D et sortie au démarrage |
| 📄 **CV intégré** | Aperçu image + zoom modal + téléchargement PDF |
| 📋 **Copie en un tap** | Email et téléphone copiables instantanément |
| 🔗 **Liens directs** | Portfolio · LinkedIn · `mailto:` · `tel:` |
| 🌐 **Open Graph** | Preview image soignée sur WhatsApp, iMessage, LinkedIn… |
| 📱 **Mobile-first** | Zones tactiles ≥ 44px · safe-area iPhone · bottom-sheet modal |
| ♿ **Accessible** | `aria-live` · `<button>` sémantiques · navigation clavier |
| 🎭 **Micro-animations** | Tilt 3D hero · ripple · shimmer · parallax CV · badges float |
| 🌙 **Anti dark-mode** | `color-scheme: only light` — palette toujours fidèle |
| ⚡ **Sans dépendances** | Vanilla HTML/CSS/JS · zéro framework · zéro build step |
